# Proyecto de prueba
Implementación proyecto para mostar pokemones obtenidos del https://localhost:8080

## Sobre este proyecto
proyecto dinamico donde se muestra la informacion de los diferentes pokemones

### Tecnologias utilizadas:
* React 
* TypeScript
* Redux
* Axios
* Material UI
* Boopstrap
